# 🤝 Contributing to MockLocationApp

Thank you for your interest in contributing! Your ideas, fixes, and tests are welcome.

---

## 🐞 Reporting Bugs

1. Go to the **Issues** tab
2. Select the “Bug report” template
3. Include:
   - Steps to reproduce
   - Expected vs. actual behavior
   - App and system version

---

## 📦 Pull Requests

1. Fork the repository
2. Create a new branch (`feature/your-feature-name`)
3. Make your changes
4. Ensure tests pass
5. Submit a Pull Request with a clear description

---

## 🎨 Code Style

- Kotlin/Java: follow Android Studio conventions
- XML: clean indentation, comments for complex layouts
- Markdown: UTF-8 encoding, use H1–H3 headers, emoji optional

---

## 💙 Get Involved

Every contribution matters — from fixing typos to adding new features.  
Join the discussions, ask questions, and help grow the project!
