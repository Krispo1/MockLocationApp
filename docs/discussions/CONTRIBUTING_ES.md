# 🤝 Contribuir a MockLocationApp

¡Gracias por tu interés en contribuir! Tus ideas, correcciones y pruebas son bienvenidas.

---

## 🐞 Informar errores

1. Ve a la pestaña **Issues**
2. Selecciona la plantilla “Bug report”
3. Incluye:
   - Pasos para reproducir el error
   - Comportamiento esperado vs. real
   - Versión de la app y del sistema

---

## 📦 Pull Requests

1. Haz un fork del repositorio
2. Crea una nueva rama (`feature/nombre-de-tu-función`)
3. Realiza los cambios
4. Asegúrate de que las pruebas pasen
5. Envía un Pull Request con una descripción clara

---

## 🎨 Estilo de código

- Kotlin/Java: sigue las convenciones de Android Studio
- XML: sangría limpia, comentarios en diseños complejos
- Markdown: codificación UTF-8, encabezados H1–H3, emoji opcional

---

## 💙 Participa

Cada contribución cuenta — desde corregir errores tipográficos hasta agregar funciones.  
Únete a las discusiones, haz preguntas y ayuda a mejorar el proyecto.
