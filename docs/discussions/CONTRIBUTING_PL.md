# 🤝 Współtworzenie projektu MockLocationApp

Dziękujemy za zainteresowanie współtworzeniem tego projektu! Twoje pomysły, poprawki i testy są mile widziane.

---

## 🐞 Zgłaszanie błędów

1. Przejdź do zakładki **Issues**
2. Wybierz szablon „Bug report”
3. Opisz:
   - Kroki prowadzące do błędu
   - Oczekiwane vs. rzeczywiste zachowanie
   - Wersję aplikacji i systemu

---

## 📦 Pull Requesty

1. Forkuj repozytorium
2. Utwórz nową gałąź (`feature/nazwa-funkcji`)
3. Wprowadź zmiany
4. Upewnij się, że testy przechodzą
5. Utwórz Pull Request z opisem zmian

---

## 🎨 Styl kodu

- Kotlin/Java: zgodnie z Android Studio Code Style
- XML: czytelne wcięcia, komentarze dla złożonych układów
- Markdown: UTF-8, nagłówki H1–H3, emoji opcjonalne
