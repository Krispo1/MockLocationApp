# 🤝 Участие в проекте MockLocationApp

Спасибо за интерес к участию! Ваши идеи, исправления и тесты приветствуются.

---

## 🐞 Сообщение об ошибках

1. Перейдите на вкладку **Issues**
2. Выберите шаблон “Bug report”
3. Укажите:
   - Шаги для воспроизведения
   - Ожидаемое vs. фактическое поведение
   - Версию приложения и системы

---

## 📦 Pull Request

1. Сделайте форк репозитория
2. Создайте новую ветку (`feature/имя-функции`)
3. Внесите изменения
4. Убедитесь, что тесты проходят
5. Отправьте Pull Request с описанием

---

## 🎨 Стиль кода

- Kotlin/Java: следуйте стандартам Android Studio
- XML: чистая структура, комментарии для сложных макетов
- Markdown: кодировка UTF-8, заголовки H1–H3, emoji по желанию
