\# 📍 MockLocationApp



MockLocationApp 是一个开源 Android 应用，用于测试 GPS 位置。支持实时位置模拟，适合开发者、测试人员和隐私爱好者。



---



\## 🚀 安装



```bash

git clone https://github.com/Krispo1/MockLocationApp.git

```



\- 在 Android Studio 中打开项目  

\- 在模拟器或设备上运行应用



---



\## 📱 使用方法



\- 在地图上选择位置  

\- 点击“Start”开始模拟  

\- 使用任何依赖位置的应用程序进行测试
